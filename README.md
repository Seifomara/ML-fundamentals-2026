# UCI Bank Marketing Model

## Description
This project uses the UCI Bank Marketing dataset, which contains data from direct marketing 
campaigns (phone calls) of a Portuguese banking institution. The goal is to predict whether 
a client will subscribe to a term deposit (variable y).

## Pipeline
1. Data Loading and Exploration
2. Data Splitting (60/20/20 stratified)
3. Managing Missing Values
4. Encoding Categorical Variables
5. Feature Scaling
6. Feature Selection
7. Addressing Class Imbalance (SMOTE)
8. Logistic Regression Training

## How to Run
1. Open the notebook in Google Colab
2. Upload `bank-additional.csv` to the Colab files panel
3. Run all cells in order (Runtime → Run all)
